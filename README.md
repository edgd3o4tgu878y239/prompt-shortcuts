# **Prompt Shortcuts (プロンプトショートカット)**
![](screenShot2026-01-18%20172455.png)

特定のトリガー文字列（デフォルト: ///）を入力するだけで、あらかじめ保存したAIプロンプトや定型文を瞬時に呼び出し・展開できるChrome拡張機能です。

プロンプトが増えても迷わない**カテゴリ管理機能**と、目的のプロンプトをすぐに見つける**高速検索機能**を両立しています。

## **✨ 主な機能**

* **クイック呼び出し**: フォームに /// と入力するだけで候補メニューを表示。
* **カテゴリ管理**: プロンプトをカテゴリごとにフォルダ分けして整理。階層メニューで視覚的にブラウズ可能。
* **高速検索**: トリガーの後に文字を入力し始めると、全カテゴリから即座に前方一致検索を実行。  
* **高度な互換性**: 通常の input, textarea はもちろん、GmailやNotionなどの contentEditable（リッチエディタ）にも対応。  
* **バックアップ機能**: 設定したショートカットをJSON形式でエクスポート・インポート可能。

## **💡 使い方**

1. **プロンプトの登録**: ツールバーのアイコンをクリックし、管理画面から「カテゴリ名」「ショートカット名」「プロンプト内容」を入力して保存します。  
2. **トリガーの設定**: 必要に応じて、呼び出し用のトリガー文字列（例: @@, :: など）を変更できます。  
3. **入力と展開**:  
   * Webサイトの入力欄で /// と入力。
     ![](screenShot2026-01-18%20172638.png)
   * カテゴリを選択するか、そのまま検索文字を入力。
![](screenShot%202026-01-18%20172615.png)
   * 目的のプロンプトを選択して Enter を押すと、その場でテキストが置換されます。
![](screenShot2026-01-18%20172654.png)
